# DIVA-jewellery
DIVA Jewellery – A sleek, modern Flask-based e-commerce web app for showcasing and purchasing minimal silver jewellery with an elegant user experience.

# Features
🛍️ Product showcase with elegant grid layout.

🛒 Add-to-cart functionality.

📦 Shopping cart with order summary.

🧾 Checkout form for user details.

🎉 Order confirmation page.

🌙 Dark-themed luxury UI with smooth hover effects

# UI Highlights
• Minimal black + silver aesthetic.

• Clean spacing and typography.

• Hover animations & soft shadows.

• Responsive product grid (4 per row)

# Tech Stack
### Backend : Python (Flask)
### Frontend : HTML, CSS, Jinja2
### Styling : CSS
